# Бот ATLANT LEX (NareBot)

Telegram-бот для клиентов юридической компании: личный кабинет по номеру телефона, заявки и вопросы в Bitrix24, FAQ, загрузка документов.

## Быстрый запуск (показать клиенту локально)

1. **Токен бота**  
   Получите токен у [@BotFather](https://t.me/BotFather), создайте бота или возьмите существующий.

2. **Настройка окружения**  
   В папке `Наре проект/NareBot`:
   ```bash
   cp .env.example .env
   ```
   Откройте `.env` и укажите:
   - `BOT_TOKEN=ваш_токен_от_BotFather`
   - `BITRIX_WEBHOOK=...` — URL вебхука Bitrix24 (если нет — бот запустится в демо-режиме: заявки/вопросы не уходят в CRM, но интерфейс работает).

3. **Зависимости и запуск**  
   ```bash
   cd "Наре проект/NareBot"
   python3 -m venv .venv
   source .venv/bin/activate   # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   python bot.py
   ```

4. В Telegram найдите вашего бота и нажмите **Старт** → **Начать**. Можно показывать клиенту: меню, FAQ, заявку, «Связаться с менеджером», личный кабинет (с CRM — по телефону; без CRM — сообщение о демо).

## Запуск из корня проекта АБФЛ

Из папки `АБФЛ` можно запустить этот бот так:

```bash
cd "Наре проект/NareBot"
source .venv/bin/activate
python bot.py
```

Или одной командой (из корня АБФЛ):

```bash
python "Наре проект/NareBot/bot.py"
```

(предварительно установите зависимости в виртуальное окружение NareBot или в общее).

## Переменные окружения

| Переменная       | Описание                          |
|------------------|-----------------------------------|
| `BOT_TOKEN`      | Токен бота от @BotFather (обязательно) |
| `BITRIX_WEBHOOK` | URL REST Bitrix24, например `https://ваш-портал.bitrix24.ru/rest/ID/CODE/` (необязательно; без неё — демо) |

## Демо-режим

Если `BITRIX_WEBHOOK` не задан:

- Личный кабинет выдаёт предложение оставить заявку.
- «Оставить заявку» и «Задать вопрос» отвечают сообщением об успехе без отправки в CRM.
- Загрузка документов сообщает, что интеграция недоступна.

Так можно локально запустить бота и показать клиенту сценарии без доступа к Bitrix24.
