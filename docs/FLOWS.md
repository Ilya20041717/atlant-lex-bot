## User Flow по ролям

### Лид
1. `/start`
2. Выбор статуса: «Я только интересуюсь»
3. Меню лида:
   - Информация
   - FAQ
   - Предварительная анкета
   - Запись на консультацию (заглушка)
4. Информация/FAQ — только общие сведения + дисклеймер.
5. Анкета — сбор данных без оценки, запись в БД.

### Клиент
1. `/start`
2. Выбор статуса: «Я уже работаю с агентством»
3. Проверка клиента в БД
4. Меню клиента:
   - Личный кабинет
   - Этапы процедуры
   - Документы (загрузка — заглушка)
   - Напоминания (автоуведомления — заглушка)
   - Финансы (интеграция платежей — заглушка)

### Сотрудник
1. `/start`
2. Выбор статуса: «Я сотрудник агентства»
3. Сообщение-заглушка, интерфейс не реализован.

### Менеджер / Юрист / Собственник
- Роли зарезервированы в системе, интерфейс не реализован.
